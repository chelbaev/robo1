Название: tree
Описание: Отображает структуру каталогов в виде дерева
Синтаксис: tree [опции] [каталог]
Пример: tree -L 2 /home/user
Примечание: -L ограничивает глубину отображения
===
Название: man
Описание: выводит мануал для команды (если есть)
Синтакисис: man [команда]
Пример: man git
===
Название: df
Описание: показывает свободное место на диске
Пример: df
===
Название: pkill
Описание: завершает запущенный процесс
Синтаксис: pkill [application_name]
Пример: pkill code
===
Название: ddate
Описание: показывает текущую дату
===
Название: rsync
Описание: качает данные с сервера на пк и наоборот
Синтаксис: rsync [OPTION]... SRC [SRC]... DEST
Пример: rsync -rz folder1 98.76.54.32:/home/user/folder2
===
Название: pwd
Описание: выводит полный путь
===
Название: grep
Описание: поиск по файлу/выводу в терминале
Синтаксис: grep [ПАРАМЕТР]… ШАБЛОНЫ [ФАЙЛ]…
Пример: ls | grep ex00
===
Название: top
Описание: выводит процессы и потребляемые ресурсы
===
Название: cp
Описание: копирует файлы/папки с места на место
Пример: cp folder1 path/to/folder2
